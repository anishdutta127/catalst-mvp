## Pass 4: AI Slop Risk — 6/10

**Classifier:** This is a HYBRID (enchanted garden marketing/brand experience with app-like interaction screens).

**Hard rejection check:**
1. Generic SaaS card grid? NO — enchanted garden theme is distinctive
2. Beautiful image with weak brand? RISK — CATALST wordmark must be strong enough to survive the dark backgrounds
3. Strong headline with no clear action? S00 is good (single name input)
4. Busy imagery behind text? RISK — Midjourney backgrounds are detailed. Vignette overlay must be aggressive enough for text readability.
5. Sections repeating same mood? NO — emotional arc varies per screen
6. Carousel with no purpose? NO — no carousels
7. App UI of stacked cards? RISK — S04 industry cards could feel like a generic list

**AI Slop blacklist check against scaffolded code:**
- ❌ GlassCard uses `backdrop-blur` glass-morphism. Glass-morphism is the 2024 AI slop equivalent of drop shadows. Every AI-generated dark UI uses glass-morphism.
- ❌ GoldButton with `box-shadow: 0 0 20px` glow. Gold glow on dark is extremely common in AI-generated fantasy UIs.
- ✓ Midjourney backgrounds are distinctive (Style B locked, not stock)
- ✓ Word-by-word streaming is intentional, not decoration
- ✓ Dark theme is justified by garden setting, not a generic choice

**Slop risk:** The glass-morphism + gold glow combination reads as "AI dark theme." The CONTENT is distinctive (Rorschach blots, personality matching, house sorting) but the VISUAL TREATMENT of UI elements needs to be more intentional.

**Re-rate: 6/10** — Midjourney backgrounds save it but UI element styling needs differentiation.

## Pass 6: Responsive & Accessibility — 3/10

**Gap:** Engineering plan has breakpoints (360px/720px) but no mobile-specific design. No accessibility spec at all.

**Fix:**

```
RESPONSIVE SPEC:

Mobile (< 640px):
  - Header: 36px, CATALST wordmark only (no milestone bar until scroll)
  - Dialogue strip: 80px max, scrollable if overflow
  - Activity: full width minus 16px padding each side
  - CTA: 56px, full width, sticky bottom
  - S04 cards: single column, full width
  - S09 ideas: stacked vertically (not side-by-side)
  - S11 card: 300x400 (Instagram story ratio)
  - Milestone bar: show current + next 2 only

Tablet/Desktop (>= 640px):
  - Header: 48px, wordmark + milestone bar
  - Dialogue strip: 15-20dvh
  - Activity: max-w-720px centered
  - CTA: max-w-720px centered, 56px
  - S04 cards: 2 columns
  - S09 ideas: 3 cards side-by-side
  - S11 card: 400x600
  - Milestone bar: show all 8 milestones

ACCESSIBILITY SPEC:

Keyboard:
  - Tab through options on all selection screens
  - Enter/Space to select
  - Escape to deselect (S06 crystal, S04 industry)
  - Arrow keys for S04 card navigation

Touch targets: 44px minimum (all buttons, options, cards)

Contrast:
  - Ivory on dark: #FAF3E0 on #0C0E12 = ratio 15:1 ✓
  - Gold on dark: #D4A843 on #0C0E12 = ratio 8:1 ✓
  - Muted text: 60% ivory on dark = verify ≥ 4.5:1

Screen readers:
  - aria-live="polite" on dialogue strip (Cedric messages)
  - aria-label on all icon-only buttons
  - role="progressbar" on milestone bar
  - role="timer" on countdown timers
  - Semantic headings per screen (h1 = screen title, h2 = sections)

Reduced motion:
  - prefers-reduced-motion: reduce → disable streaming animation,
    show text instantly, disable background crossfade, disable
    selection dissolve
```

**Re-rate: 7/10** — responsive + a11y specs added. Missing: detailed ARIA states for complex widgets (S04 filter chips, S06 orb grid).
